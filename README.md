# Elegant Context

An Online Shopping Application made using React.

## Run Locally

Clone the project

```bash
  git clone https://github.com/mclods/elegant-context
```

Go to the project directory

```bash
  cd elegant-context
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Test using cypress

```bash
  npm run cypress
```
